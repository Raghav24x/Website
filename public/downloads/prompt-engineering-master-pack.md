## Section 1 — Financial Analysis Prompts

**1.1 Net Worth Snapshot**
```
You are a personal finance analyst. Given the following assets and liabilities, calculate my net worth, identify my top 3 wealth-building opportunities, and flag any risk concentrations. Be direct and specific.

Assets: [list]
Liabilities: [list]
```

**1.2 Investment Thesis Builder**
```
Act as a sell-side analyst. Given the following company [NAME], write a structured investment thesis covering: bull case, bear case, key risks, valuation range, and a 12-month price target rationale. Use publicly available data up to your knowledge cutoff.
```

**1.3 Portfolio Stress Test**
```
Stress-test this portfolio against three scenarios: (1) 2008-style credit crisis, (2) 2022 rate-hike environment, (3) AI-driven sector disruption. For each, estimate percentage drawdown and time to recovery. Portfolio: [list holdings + weights].
```

**1.4 Creator Revenue Optimisation**
```
I run a [newsletter/YouTube/course] with [X] free subscribers and [Y] paid at [$Z/mo]. Analyse my revenue model, identify my biggest monetisation gaps, and give me 3 specific tactics to grow MRR by 20% in 90 days.
```

## Section 2 — AI Product & Research Prompts

**2.1 AI Tool Due Diligence**
```
Evaluate [TOOL NAME] as a potential addition to my AI stack. Cover: core capabilities, pricing model, integration complexity, alternatives, and vendor risk. Score it 1–10 overall and give a clear Buy / Trial / Avoid verdict.
```

**2.2 LLM Selection Framework**
```
I need to choose an LLM for [USE CASE]. My constraints: [budget/latency/privacy requirements]. Compare the top 3 candidates across capability, cost per 1M tokens, context window, and deployment options. Recommend one with justification.
```

**2.3 System Prompt Audit**
```
Review this system prompt for a [CHATBOT/AGENT/ASSISTANT]. Identify: (1) ambiguities that could cause inconsistent outputs, (2) missing guardrails, (3) prompt injection risks, (4) ways to make it 30% more concise without losing intent.

System prompt: [PASTE HERE]
```

**2.4 RAG Query Optimisation**
```
I have a RAG pipeline returning poor retrieval quality for these query types: [examples]. Diagnose the likely causes (chunking, embedding model, retrieval strategy, query rewriting) and propose 3 specific fixes with expected impact.
```

**2.5 AI Agent Task Decomposition**
```
Break down this complex task into a multi-step agent plan: [TASK]. For each step specify: (1) the sub-task, (2) the tool or capability needed, (3) the expected output, (4) error handling strategy. Format as a numbered plan I can implement in code.
```

## Section 3 — Business & Startup Prompts

**3.1 Runway Calculator Prompt**
```
Given: monthly burn $[X], current cash $[Y], MRR $[Z] growing at [%]/month. Calculate: (1) runway at current burn, (2) runway if I cut 20%, (3) month I hit break-even, (4) how much I need to raise for 18 months of runway. Show your working.
```

**3.2 Pricing Strategy**
```
I'm pricing a [PRODUCT] for [AUDIENCE]. Current options I'm considering: [list tiers/prices]. Analyse using value-based pricing principles. Suggest the optimal pricing structure, justify each tier, and predict which tier most users will choose.
```

**3.3 Cold Outreach Personalisation**
```
Write a personalised cold email to [NAME/ROLE] at [COMPANY] about [OFFERING]. Use this public context about them: [LinkedIn summary / recent post / company news]. Keep it under 100 words. No generic openers. End with a specific, low-friction CTA.
```

## Section 4 — Content & Newsletter Prompts

**4.1 Hook Generator**
```
Generate 10 newsletter hooks for this topic: [TOPIC]. Each hook should be under 20 words, lead with a counterintuitive claim or surprising stat, and be written for an audience of AI builders and financially curious founders.
```

**4.2 Thread-to-Newsletter Expander**
```
Expand this tweet thread into a 700-word newsletter section. Maintain the direct, punchy tone but add: context, one concrete example, one relevant data point, and 3 actionable takeaways. Thread: [PASTE]
```

**4.3 TL;DR Summariser**
```
Summarise this [article/paper/report] in exactly 5 bullet points, each under 20 words. Prioritise: (1) the main finding, (2) what changed vs prior understanding, (3) the number that matters most, (4) who this affects, (5) what to do about it.
```
